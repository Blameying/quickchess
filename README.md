### QuickChess

因与友人外出手中无棋盘，又觉现有下棋软件太过繁琐，注册，充值，广告。便想写一个棋盘
借此机会尝鲜使用flutter，因其跨平台性和热重载调试比较舒服，且不需要大型IDE。

目前只是五子棋，可以观战，黑白子系统分配。

服务端目前用的是sinatra+redis,房间定时销毁，暂未上传
